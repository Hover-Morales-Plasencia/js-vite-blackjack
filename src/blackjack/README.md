# Es el primer cambio que se realizara para que funcione git-hub

Pasos para subir al GIthub
**git init** Convierta un directorio existente en un repositorio git
**git add .** Instantáneas del archivo en preparación para el control de versiones
**git commit -m "(mensaje)"**  Registra instantáneas de archivos de forma permanente en el historial de versiones.
**git checkout -- .** reconstruye el ultimo commit

**Luego de esto vienen los comandos del propio github pra subir a la nube**
